toQuiz
Mobile-optimized quiz tool for adding an element of gamification to news.
Introduction

toQuiz is a reusable, open-source tool that can help newsrooms introduce basic gamification to a story and create more online engagement around a topic. It can also be used for creating weekly news wrap-ups that help newsrooms to highlight recent important or interesting content for readers.

The tool is mobile first and fully responsive on different device sizes. It is optimized for quick loading on article pages and will not slow down your website. Unlike most embeddable tools that don’t benefit a story’s SEO, content on each of the toQuiz cards is SEO friendly, meaning that it is fully crawlable by search engines.

toQuiz is available to journalists using ICFJ partner organization Pykih’s PhotoGraph platform. Email ICFJ Knight Program Manager Sara Olstad (solstad@icfj.org) to get started.
